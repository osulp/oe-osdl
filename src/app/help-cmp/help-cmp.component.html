How to Publish Data:

Is your dataset an element of Oregon's Framework Program?

  • Yes: Requests to upload framework data elements should go to Theresa Burcsu, DAS Framework Coordinator (theresa.burcsu@oregon.gov).

    Framework data should include metadata that meets the Oregon Geospatial Metadata Standard. See the Resources page for more information about creating metadata that conforms to the Oregon Standard.

  • No: Non-framework data should meet the collection guidelines (listed below), and include at least the minimum metadata requirements (also listed below). It is recommended that the metadata meet the Oregon Geospatial Metadata Standard, but that is not a requirement for publication.

    To publish non-framework data contact virtualoregon.support@oregonstate.edu.

    • Collection Guidelines:

      ·   Data and metadata are in their final form.

      ·   Metadata complies with OSDL minimum metadata requirements.

      ·   All file names and attributes are well described and interpretable. Any acronyms used are defined in the metadata.

      ·   The data has potential value for other application areas or to support decision making.

      ·   The geographic extent is substantial (approximately the size of an average county or 4th field HUC (700 square miles)), or the extent covers a city.

       

    • Minimum Metadata Requirements:

      ·  Title (Overview>Item Description>Title)

      ·  Description/Abstract (Overview>Item Description>Description (Abstract))

      ·  Topic Categories/ISO Theme (Overview>Topics&Keywords>Topic Categories)

      ·  Metadata Contact (Metadata>Contacts>New Contact>New Contact Information (set role to “Originator”))

      ·  Publish Date (Overview>Citation>Dates>Published)


How to Download Data:
